## Installation

### Dependencies and Versions

Node version: v8.12.0
yarn version: v1.9.4

### Installing Dependencies

When you have the correct versions, install the required dependencies:

```
yarn install
```

### Running the project

```
yarn start
```

### Running tests

```
yarn test
```
